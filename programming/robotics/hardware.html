RoboRIO Setup

1Initial RoboRIO Configuration

  1. Connect the RoboRIO to your computer via USB
  2. Open the roboRIO Imaging Tool
  3. Select the latest firmware image
  4. Enter your team number
  5. Click "Reformat" and "Install"
Reformatting will erase all existing code and configurations!

2Network Configuration

Static IP Configuration

Configure the following network settings:

  • IP Address: 10.TE.AM.2
  • Subnet Mask: 255.255.255.0
  • Gateway: 10.TE.AM.1

Power Distribution Hub Setup

1PDH Installation

  1. Mount PDH securely to robot frame
  2. Connect main power cables from battery
  3. Connect CAN cable to RoboRIO
  4. Install 120A main breaker

2Power Distribution

Recommended Power Layout:
Port Device Breaker Size
0-3 Drive Motors 40A
4-7 Mechanism Motors 30A
8-11 Small Motors 20A

Motor Controller Setup

1SPARK MAX Configuration

  1. Connect SPARK MAX to USB
  2. Open REV Hardware Client
  3. Update firmware if needed
  4. Set CAN ID (unique for each controller)
  5. Configure motor type (NEO, NEO 550, etc.)

Testing & Verification

1System Verification

Test Sequence:
Detailed Test Procedures
1. Power Up System Check
  • Verify battery voltage (>12V)
  • Check main breaker connection
  • Confirm RoboRIO power LED
  • Verify PDH status lights
2. Communication Verification
  • Test radio connectivity
  • Verify CAN bus communication
  • Check Driver Station connection
3. Motor Movement Test
  • Test each motor individually
  • Verify correct direction
  • Check encoder feedback
4. Sensor Operation
  • Test limit switches
  • Verify gyro initialization
  • Check distance sensor readings
